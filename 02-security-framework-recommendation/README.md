# Security Framework Recommendation
### NIST CSF 2.0 vs ISO 27001/2 — Framework Analysis for a Small E-Commerce Retailer

**Prepared by:** Alister A. Rodrigues — Security Analyst, Obsidian SecureTech Solutions
**Client:** Crestline Apparel Co. | **Date:** September 2024

---

## Overview

Crestline Apparel Co., a small US-based e-commerce retailer, needed a cybersecurity framework recommendation ahead of scaling its digital operations. This analysis evaluates **NIST CSF 2.0** against **ISO/IEC 27001/2** across eight criteria — flexibility, global applicability, certification, risk management focus, cross-referencing, cost, implementation complexity, and ongoing maintenance.

---

## Recommendation

**NIST CSF 2.0** — recommended as the primary framework at this stage.

NIST CSF 2.0's incremental adoption model allows Crestline to build security maturity proportionally without the overhead of formal ISMS certification. It maps cleanly to PCI DSS, CCPA, and GDPR obligations relevant to the e-commerce context. ISO 27001/2 is identified as the natural progression path as the business scales internationally.

---

## Scope

- Feature and limitation analysis of NIST CSF 2.0 and ISO 27001/2
- Eight-criteria comparative table
- Regulatory compliance mapping: CCPA, PCI DSS, GDPR
- Phased adoption recommendation
